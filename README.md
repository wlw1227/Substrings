# Substrings
 Finds a given substring in an array and returns the amount of occurances. 

 Takes two parameters in the #substrings method, the word or phrase, and the dicitonary for comparision. The #substrings method will count the amount of times any words in the dictionary array appear in the given word or in each word of the given phrase. 

 The return of the #substrings method is a hash with the found dicitonary word as the key and the number of occurances as the value.
